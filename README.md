# House Price Tableau

## Introduction

The **House Price Tableau** project aims to analyze and visualize housing prices data using Tableau. This repository contains visualizations and insights derived from data to help understand key trends and patterns in house prices.

---

## Features

- Interactive dashboards to explore housing price data.
- Insights into factors affecting housing prices.
- Easily accessible visualizations for non-technical stakeholders.

---

## Prerequisites

Before exploring this project, ensure you have:

- Tableau Public or Tableau Desktop installed.
- Access to the dataset used in the analysis.

---

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/mahwishhina/House_Price-Tableau.git
   ```
2. Open the visualization file (`.twb/.twbx`) in Tableau.
3. Explore and interact with the dashboards to uncover insights.

---

## Dataset

The dataset used for this project is included or linked within the repository. For custom analysis, any housing price dataset following similar attributes and structure can be used.
